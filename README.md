# Battle Lab 1914 - Etapa 18 UX Simple

## Objetivo

Esta etapa reduce la saturación visual.

La calculadora ahora abre en modo simple para que el usuario entienda la herramienta en segundos.

## Tres modos

### Modo básico

- Army A.
- Army B.
- Unidad.
- Cantidad.
- HP.
- Simular.
- Resultado claro.

### Modo táctico

Agrega:

- posición,
- terreno,
- tipo de objetivo,
- objetivo del stack.

### Modo avanzado

Agrega:

- fortalezas,
- HP de fortaleza,
- héroes,
- bonos,
- varianza,
- exportación.

## Resultado simplificado

El resultado principal ahora aparece arriba, con:

- ganador estimado,
- HP restante,
- alertas claras,
- resumen táctico.

El detalle técnico queda escondido en una sección desplegable.

## Escenarios rápidos

Incluye:

- desembarco urgente,
- bombardeo a fortaleza,
- ataque aéreo,
- Georg + artillería.
